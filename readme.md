# Team Hub

## How to install

You will need
  - node.js:http://nodejs.org/,
  - grunt-cli: http://gruntjs.com/getting-started
  - ruby: https://www.ruby-lang.org

```
cd server
bundle install
cd ../client
npm install
```

## How to run

To run the server

```
cd server
./script/server
```

To copy the dist to the server and start the watch

```
cd client
grunt
```
